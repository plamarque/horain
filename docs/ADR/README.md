# Architecture Decision Records (ADR)

Un fichier par décision, nommé `NNNN-titre-court.md` (ex. `0001-choix-framework.md`).

Format suggéré :

```markdown
# ADR-NNNN : Titre

## Contexte
[Pourquoi la décision est nécessaire.]

## Décision
[Ce qui a été décidé.]

## Conséquences
[Avantages, inconvénients, impacts.]
```
