# ADR-0001 : Stack front-end Vue 3, PrimeVue, Vite

## Contexte

Le projet doit fournir une interface utilisateur mobile-first, PWA, avec une bonne ergonomie et une maintenance facilitée.

## Décision

- **Framework :** Vue 3
- **UI :** PrimeVue
- **Build :** Vite
- **Langage :** HTML, CSS, JavaScript/TypeScript

## Conséquences

- PrimeVue fournit des composants prêts à l’emploi pour une UI cohérente.
- Vite assure un build rapide et une DX fluide.
- Alignement possible avec l’écosystème Vue (Composition API, Pinia si besoin).
